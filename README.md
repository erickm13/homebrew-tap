# homebrew-tap

Homebrew tap for [Salchipapa.Dots](https://github.com/erickm13/Salchipapa.Dots) — a Linux dotfiles setup for Fish/Zsh, Zellij/Tmux, Neovim & CLI tools.

## Install

```bash
brew tap erickm13/tap
brew install salchipapa-dots
```

Then run the installer:

```bash
sudo salchipapa-dots
```

## What's included

| Tool | Options |
|---|---|
| Shell | Fish, Zsh |
| Multiplexer | Zellij, Tmux |
| Editor | Neovim (LazyVim) |
| Terminal | Alacritty, Wezterm |
| Prompt | Starship |
| CLI tools | zoxide, atuin, fzf, carapace |

## Formulas

| Formula | Description |
|---|---|
| `salchipapa-dots` | Dotfiles installer |

## Update

The formula is automatically updated on every new release of [Salchipapa.Dots](https://github.com/erickm13/Salchipapa.Dots) via GitHub Actions.
